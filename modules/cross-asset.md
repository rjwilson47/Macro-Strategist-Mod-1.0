# Module: Cross-Asset Synthesis

**Depends on:** All asset-class modules that have been completed, Working brief
**Feeds into:** Scenarios (probability-weighted returns across assets), Actionable Conclusions

*Activate for any cross-asset, relative value, or broad allocation report. Also include a lighter version in any report where findings in one asset class have material implications for another.*

---

## Core Framework

- **Relative valuation:** Compare expected returns across asset classes using consistent metrics — earnings yield, credit spread, real yield, carry. Apples-to-apples where possible.
- **Correlation regime:** Stocks-bonds correlation positive or negative? What regime are we in? Implications for diversification.
- **Risk budgeting:** Where is the best risk-adjusted return opportunity? Sharpe-like thinking across assets.
- **Diversification assessment:** Are traditional diversifiers (bonds, gold, alternatives) working?
- **Regime identification:** Map current environment to a historical regime (e.g., 2003 recovery, 2017 goldilocks, 2022 rate shock) — what performed in analogous periods? Only include if the analogy is genuinely informative.
- **Capital flow analysis:** Where is capital migrating? (e.g., equities → money markets, EM → DM)

---

## Working Brief Update

This module typically runs after most other analytical modules. Its role in the brief is to **synthesise and resolve tensions** — e.g., if equities look cheap but credit is pricing recession risk, which signal dominates?
