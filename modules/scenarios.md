# Module: Scenarios & Probability Framework

**Depends on:** Working brief (all analytical findings)
**Feeds into:** Production (scenario table), Actionable Conclusions (probability-weighted returns)

*Include in every report. The format adapts to the topic.*

---

## Format by Report Type

### Directional Asset Class Views
Bull / Base / Bear with explicit probability weights:

| Scenario | Probability | Description | Target/Return | Key Driver |
|---|---|---|---|---|
| Bull | X% | ... | ... | ... |
| Base | Y% | ... | ... | ... |
| Bear | Z% | ... | ... | ... |

*Probability-weighted expected return = Σ(return × probability)*

### Relative Value / Cross-Asset Views
Upside/downside sizing:
- Expected return in base case
- Upside scenario: magnitude and probability
- Downside scenario: magnitude and probability
- Risk/reward ratio (probability-adjusted)

### Thematic Views
Scenario tree: 2–3 branching outcomes (e.g., "capex accelerates" vs "capex disappoints" vs "regulation constrains") with probability and asset class implications for each branch.

---

## Probability Assignment Guidance

- Subjective estimates informed by data — not precise forecasts. State this.
- Use round numbers (50/30/20, not 47/31/22).
- Must sum to 100%.
- If quantification is genuinely impossible, state scenarios qualitatively and explain why.

---

## Key Indicators to Watch (mandatory)

Every report must include 5–10 indicators in this format:

| Indicator | Current Level | Confirming Signal | Disconfirming Signal | Next Release/Trigger |
|---|---|---|---|---|
| [Name] | [Value, sourced] | What confirms thesis | What challenges/invalidates | Date |

**Be specific about thresholds.** "Watch ISM" is useless. "ISM >52 confirms re-acceleration; <48 signals contraction risk" is actionable.

Include dates where possible. This section answers: *"Between now and my next report, what data would make me change my mind?"*

### Indicator Types by Report

- **Equity:** Earnings revision breadth, key macro releases with thresholds, valuation triggers, positioning thresholds, bellwether earnings dates, policy calendar
- **Fixed income:** CPI/PCE with thresholds, central bank dates + what's priced, spread reassessment levels, auction results, curve slope thresholds
- **Macro:** Leading indicators with thresholds, FCI levels, credit growth, labour market, confidence surveys
- **Thematic:** Theme-specific data (capex, chip sales, procurement contracts), regulatory milestones, key company earnings dates
- **Cross-asset:** Correlation breakpoints, relative valuation thresholds, flow reversal signals, regime change indicators
